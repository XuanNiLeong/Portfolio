Mentor & Mentee Web-Based System

This project is not solely an assignment for my Software Engineering module but it is the project that sparked my interest in Software Development Project and unfolded opportunities in a Software Engineering career. I assisted my “client” in developing and deploying a mentor & mentee web-based system that will allow university students to scout for potential mentors who are industry professionals.

Insights

The system was delivered to the client over two iterations, as per an agile software development strategy. From planning to testing and integration, I have the opportunity to experience almost all of the phases in the software development life cycle, which is the application of standard business practices in building software applications ( unfortunately not for the maintenance phase) .

My team and I started off with producing a requirement document to identify the system’s functionalities and had a few rounds of client meetings to finetune the requirement document.

During the implementation phase, I was mainly responsible for managing the interaction of databases - for example, linking the registration database to the login database and the search database. Besides, I was also responsible for conducting unit testing and acceptance testing to ensure the correctness of the system's functionalities.

Tech Stack

  • Ruby
    A high-level programming language used to develop this web-based system.
  • SQLite
    A simple, self-contained and little to no configuration required relational database management system used to manage the interaction of databases.
  • HTML
    A hypertext markup language used to design the basic layout and formatting of Web pages.
  • CSS
    A language for describing the presentation of Web pages, including colors, layout, and fonts.
  • Codio
    A browser-based IDE used to develop this web-based system.
